Firmware Web Installer

WDGWatch donanımınızı doğrudan tarayıcınızdan güncelleyin veya sıfırlayın.

⚠️ Başlamadan Önce:

  • Cihazınızı USB-C kablosuyla bilgisayara bağlayın.
  • Google Chrome, MS Edge veya Opera gibi Web Serial API destekleyen bir tarayıcı kullanın.
  • Başka bir programın (PlatformIO, Arduino IDE veya Seri Monitör) portu meşgul etmediğinden emin olun.
Tarayıcınız Web Serial API desteklemiyor. Lütfen Chrome veya Edge kullanın.
